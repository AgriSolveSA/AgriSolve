# Pack Enablement Checklist (Quick)

For any client engagement:
1) Identify client type: Broker / MGA / Underwriter
2) Select packs (A–E)
3) Confirm minimum exports for selected packs
4) Build only those facts/dims
5) Load sample exports (clean + dirty)
6) Validate:
   - policy mapping completeness %
   - broker mapping completeness %
   - premium/commission coverage %
   - claims reserve availability (if needed)
7) Deploy the pack pages (wireframes drive scope)
8) Lock scope in the SOW: “Only packs selected, anything else is change request”
