# On-Prem Data Gateway Checklist (Client-Hosted)

- [ ] Gateway installed on always-on machine (not a laptop)
- [ ] Gateway service account documented
- [ ] Network access to data sources confirmed
- [ ] Gateway cluster (optional) if uptime-critical
- [ ] Dataset bound to correct gateway
- [ ] Refresh schedule set and tested
- [ ] Alerting configured (email on refresh failure)
